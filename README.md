# WebMart Seller Center

1. Clone the repository
   ```bash
   git clone [your-repository-url]
   cd webmart-seller
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser
